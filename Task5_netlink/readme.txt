Задание 5
Написать модуль ядра для своей версии ядра, который будет обмениваться информацией с userspace через netlink.

Взаимодействие между пространством ядра и пользователся при помощи сырых сокетов netlink.
User отправляет сообщение в ядро.
Ядро принимает (записывает в логи) и посылает ответное сообщение
User принимает сообщение и выводит нак экран.
